MPEGelib Example App

To try and show you how to use MPEGelib I have written a small Application to make MPEG movies from a sequence of ppm files (and a very small subset of ppm images at that)

The example app is in the directory libexample and comes with some sample images to turn into an MPEG movie for you

The calling convention for the app is
simple_encode

check out the source code to work out how it works

to Make the movie from the example images first compile the program (after editing the makefile) and then type
./simple_encode 0 15 data/pic test.mpg
This code is really just to show you how to use the library and fill up an ImVfb. The ppm reading stuff is rubbish and probably only reads in the images provided. If you want to make an MPEG movie of ppm images (or any other format) then of course you should be using Berkeley MPEG encoder
