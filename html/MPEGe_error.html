MPEGe_error

extern char *MPEGe_error;

if one of the functions

  • MPEGe_open
  • MPEGe_image
  • MPEGe_close MPEG sequence
returns FALSE (i.e. failure) then the string MPEGe_error will contain a small description of what went wrong (you may find this useful for debugging)
