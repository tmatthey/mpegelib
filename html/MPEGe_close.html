MPEGe_close

int MPEGe_close( void );

Once you have added all the images you want in your MPEG stream this function will reset the library to a sane state and create the MPEG end sequences and close the output file

After this function has been called your finished MPEG stream will be waiting for you to play with it :-)

The function returns non NULL (i.e. TRUE) on success and Zero (or FALSE) on failure.
