MPEGe_ImVfbAlloc

ImVfb *MPEGe_ImVfbAlloc(int width,  int height,
                        int fields, int for_image );

This function is designed to allow to easily allocate memory for an ImVfb. It is basically a copy of ImVfbAlloc from the SDSC Image Library with an extra added argument for_image if this is TRUE then memory is allocated to store the image, otherwise only memory for the ImVfb struct is allocated (useful for just using the ImVfb to index into your own arrays)

To find out more about ImVfb's check out my Image types page
