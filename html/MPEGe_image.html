MPEGe_image

int MPEGe_image( ImVfb *image );

Calling this function appends the given image to the MPEG stream. The format of the image pointed to by image is that used by the SDSC Image library (See my page on image types to see how to specify this. You do not need this library to use MPEGelib, but it's such a good library it's worth looking at.

The function returns non NULL (i.e. TRUE) on success and Zero (or FALSE) on failure.
