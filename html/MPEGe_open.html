MPEGe_open

int MPEGe_open( FILE *mpegfile, MPEGe_options *options );

This is the initialisation function which must be called before any of the others in the MPEGe library. There are only 2 arguments and the second is optional

mpegfile
This should be a non-NULL file pointer to the file you wish the MPEG stream to be output to.
options
This allows you to specify how the MPEG encoder does it's encoding. If this is NULL then the default options are used. See my page on options for more info.

The function returns non NULL (i.e. TRUE) on success and Zero (or FALSE) on failure.
